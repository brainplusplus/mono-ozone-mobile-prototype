<static> Mono.Caching.GetDataFromUrl(url, method, params, callback)
Gets data from a URL. If done on a native device, the data is cached. If the device goes offline, the data will be retrieved from the cache.
Parameters:
{string} url
The URL to retrieve.
{string} method
Either GET or POST
{Object} params
Parameters to pass to a URL via POST. Should be a regular JSON object.
{string} callback
A callback function to get the results.
  • status {string}: "success" for sucess, "failure" for failure.
  • message {string}: Empty on success, otherwise populated with the error message
  • data {string}: The data retrieved from the URL.

<static> Mono.Caching.Status(url, callback)
Retrieves status of data stored on the device side cache.
Parameters:
{string} url
The url where the data was saved to (see Mono.Caching.Store)
{string} callback
A callback function to get the results.
  • status {string}: "success" for success, "failure" for failure.
  • url {string}: The URL requested (for convenience)
  • timedOut {boolean}: Whether or not the data has timed out
  • timedOut {long}: When the data times out (milliseconds)
  • timeOutDate {Date}: When the data times out (Date)
  • expired {boolean}: Whether or not the data has expired
  • expiration {long}: When the data expires (milliseconds)
  • expirationDate {Date}: When the data expires (Date)
Throws:
{IllegalAccessException}
Thrown if this function is called by a desktop application.
