<static> Mono.Modals.ShowYesNoModal(title, message, callback)
Displays a yes/no modal on the screen. 
Example:
var cbk = function(data){
      var jsonDiv = document.getElementById('jsonDiv');
      jsonDiv.innerHTML= "Response: " + JSON.stringify(data) ;
 }
Mono.Modals.ShowYesNoModal(title,message,cbk);
Parameters:
{string} title
The title of the yes/no modal.
{string} message
The message to display in the modal.
{?function(({outcome: string|status: string|message: string}|{status: string|message: string}))} callback
A callback function to get the results. Must take one argument data.
{Object} callback.data Optional
  • status {string}: "success" for success, "failure" for failure.
  • message {string}: Empty on success, otherwise populated with the error message
  • outcome {string}: The outcome of the modal. "Yes" if the user has clicked Yes, "No" if the user has clicked No.
Throws:
{IllegalAccessException}
Thrown if this function is called by a desktop application.
