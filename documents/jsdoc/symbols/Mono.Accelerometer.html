<static> Mono.Accelerometer.DetectOrientationChange(detectionCallback, callback)
Executes a callback when the device orientation has changed. 
Example:
     function detectOrientation(data){
                   $("#orientation").html(data.status+"
");
                    $("#orientation").append(data.message+"
");
                    $("#orientation").append(data.orientation);
                }

     $(document).on('click', 'button[value="registerOrientation"]', function(event){
                        Mono.Accelerometer.DetectOrientationChange(detectOrientation);
                        });
Parameters:
{?function(({orientation: string|status: string|message: string}|{status: string|message: string}))} detectionCallback
callback A callback function to get the results. Must take one argument data.
{?function({status: string|message: string})} callback
A callback function to be called upon completion of the rest call.
{Object} callback.data Optional
  • status {string}: "success" for success, "failure" for failure.
  • message {string}: Empty on success, otherwise populated with the error message
  • orientation {string}: Returns the new orientation that the device has moved to. Can be one of the following:
    • portrait
    • upsideDown
    • landscapeLeft
    • landscapeRight
Throws:
{IllegalAccessException}
Thrown if this function is called by a desktop application.

An object with a member "type." Type must be "orientation" to unregister from DetectOrientationChange updates and "accelerometer" to unregister from RegisterForUpdate updates.

<static> Mono.Accelerometer.RegisterForUpdate(settings, detectionCallback, callback)
Registers for accelerometer updates from the native device. 
Example:
     function updateACC(data){
         $("#info").html(data.status+"
")
         $("#info").append(data.message+"
");
         $("#info").append(new Date().toLocaleTimeString());
         if(data.acceleration){
             $('#xCoord').html(data.acceleration.x);
             $('#yCoord').html(data.acceleration.y);
             $('#zCoord').html(data.acceleration.z);
         }
      }
      $(document).on('click', 'button[value="acc"]', function(event){

         Mono.Accelerometer.RegisterForUpdate({interval:7},updateACC);
      });
Parameters:
{{interval: int}} settings
The interval, in seconds, between callback executions.
{?function(({acceleration:{x: double|y: double|z:double}|status:string|message:string}|{status: string|message: string}))} detectionCallback
A callback function to get the results. Must take one argument data.
{?function({status: string|message: string})} callback
A callback function to be called upon completion of the rest call.
  • status {string}: "success" for success, "failure" for failure.
  • message {string}: Empty on success, otherwise populated with the error message
  • acceleration {Object}: Has information about acceleration along 3 axes.
    • x {double}: Acceleration along the x axis.
    • y {double}: Acceleration along the y axis.
    • z {double}: Acceleration along the z axis.
Throws:
{IllegalAccessException}
Thrown if this function is called by a desktop application.
